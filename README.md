# 🐄 PashuSeva Platform

PashuSeva is a digital platform designed to provide veterinary and animal care services online. The platform connects animal owners with veterinarians, enabling easy appointment booking, consultation, and access to animal healthcare services.

---

## 🚀 Features

### 👤 User Features
- 📝 User Registration & Login (JWT Authentication)
- 🐕 Add & Manage Animal Profiles
- 📅 Book Veterinary Appointments
- 📍 Location-based Service Access
- 💬 Chat / Consultation with Doctors (Future Scope)
- 📜 View Appointment History

### 👨‍⚕️ Doctor Features
- 🩺 Doctor Registration & Login
- 📋 Manage Profile (Specialization, Experience, Fees)
- 📅 Accept / Reject Appointments
- 📊 View Patient Details

### 🛠 Admin Features (Future Scope)
- 🧑‍💼 Manage Users & Doctors
- 📊 Analytics Dashboard
- 🚫 Block/Approve Accounts

---

## 🏗️ Tech Stack

### 🔹 Frontend (Android App)
- Kotlin
- Jetpack Compose
- Material UI (Compose)
- Navigation Component

### 🔹 Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- REST API Architecture

### 🔹 Tools & Libraries
- Retrofit / OkHttp (API Calls)
- Gson / Moshi (JSON Parsing)
- bcrypt (Password Hashing)
- dotenv (Environment Variables)
